# 🚀 TaskFlow - MERN Task Manager

TaskFlow is a full-stack MERN Task Management application that helps users organize and manage their daily tasks. Users can create, update, delete, search, and filter tasks while viewing real-time dashboard statistics.

---

## 🌐 Live Demo

**Frontend:**  
https://taskflow-mern-delta.vercel.app

**Backend:**  
https://taskflow-server-uk5x.onrender.com

## ✨ Features

- Create new tasks
- Update existing tasks
- Delete tasks
- Search tasks by title
- Filter by status
- Filter by priority
- Dashboard statistics
- Task completion progress
- Responsive UI

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/fayazshaik1302/taskflow-mern.git
```

### Navigate to project

```bash
cd taskflow-mern
```

### Install backend dependencies

```bash
cd server
npm install
```

### Install frontend dependencies

```bash
cd ../client
npm install
```

---

## ▶️ Run Backend

```bash
cd server
npm start
```

---

## ▶️ Run Frontend

```bash
cd client
npm run dev
```
